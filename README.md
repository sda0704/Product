Товароведение - товары со скидкой и без.

Задание 1.
Написать класс Product, описывающий товар (Наименование, Производитель, Цена, Срок годности, Дата производства). Написать программу, демонстрирующую использование этого класса. Программа должна запрашивать информацию с консоли и выводить на консоль результат. Необходимо переопределить ToString у класса Product.

Задание 2. 
На основе класса Product из предыдущего задания создать класс DiscountedProduct, описывающий товар со скидкой. Дополнить класс свойствами «размер скидки (%)» и «акционная цена».

Стек технологий:

C# / .NET
Консольное приложение без сторонних библиотек

Функциональность:

1. Создание товара
1.1 Создание товара с акционной ценой
1.2 Создание товара без акционной цены
2. Валидация введенных данных
3. Просмотр информации о товаре

Структура проекта:

Product.cs - базовый класс.
DiscountedProduct.cs - наследник.
Program.cs - точка входа в программу + логика.
